# Sign Language Translation System

A computer vision-based Sign Language Translation System that recognizes hand gestures and translates them into text in real time.

## 📌 Project Overview

This project aims to bridge the communication gap between hearing-impaired individuals and non-sign-language users by translating sign language gestures into readable text using machine learning and computer vision techniques.

The system captures hand gestures through a webcam, processes the images, extracts relevant features, and predicts the corresponding sign language character or word.

## ✨ Features

- Real-time hand gesture detection
- Sign language recognition using Machine Learning
- Webcam-based input
- Dataset collection module
- Data preprocessing and training pipeline
- Text output generation
- Easy-to-use Python implementation

## 🛠 Technologies Used

- NumPy

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/s4mud/Sign-Language-Translation-System.git
cd Sign-Language-Translation-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ Usage

Run the main application:

```bash
python final.py
```

Allow webcam access when prompted.

## 🔄 Workflow

1. Collect sign language images using the dataset collection module.
2. Train the model on the collected dataset.
3. Start the translation system.
4. Show hand gestures to the webcam.
5. Receive translated text output.
